# 骑手女王 — 角色档案

> 状态：已确认。后续创作中以本文件为角色圣经。

## 主要角色

### 苏冉（女主角 / "骑手女王"）

| 维度 | 设定 |
|------|------|
| 年龄 | 28 岁 |
| 外貌 | 165cm，中上长相，常年风吹日晒皮肤不算白但很干净。扎马尾戴头盔时是"苏师傅"，下班放下头发换掉工服后判若两人 |
| 学历 | 中国人民公安大学 犯罪心理学硕士 |
| 前职业 | 市公安局危机谈判组，最年轻的谈判组长。处理过跳楼、劫持、家暴对峙。队内绰号"冰箱"——不管对面多炸，她永远是全场最冷静的那个人 |
| 离职原因 | 最后一起劫持案，她成功说服嫌疑人放下刀——但上级指挥失误，人质受伤。舆论翻车后她被推出来背锅，"个人原因主动离职"。她再也没对人讲过那天的事 |
| 入行契机 | 失业半年，面试 37 家公司全挂——"你学历太高了我们不敢用"、"谈判专家？我们又不是黑社会"。房贷还剩 23 年。看到饿了么招聘广告写着"月入过万，学历不限"，心想先跑一个月 |
| 性格 | **外在**：嘴贫、脸皮厚、什么场面都稳得住。能用专业术语解构一切荒唐事。**内在**：要强、不服输、对不公平零容忍。把自己活成一座孤岛——能化解所有人的冲突，唯独不处理自己的创伤 |
| 核心能力 | 读人（30 秒判断人格类型）、控场（语速/节奏/沉默的精准使用）、降级（让对方在不知不觉中自己把情绪降下来）、BATNA 分析（任何情况下都知道自己有什么底牌） |
| 口头禅 | "别激动，我们先明确一下各自的诉求。" （对谁都像在谈判） |
| 喜剧锚点 | **过度专业 x 鸡毛蒜皮**——用解救人质的专注度和技术来处理"外卖少放香菜"级别的纠纷。内心独白全是威胁评估模型和谈判策略分析，嘴里说的全是"您消消气" |
| 致命弱点 | 1) 妈妈——所有谈判技巧在血脉压制面前清零 2) 真诚——遇到完全不需要谈判的"正常人"反而手足无措 3) 不擅长求助——能帮所有人搞定问题，唯独不会让人帮自己 |
| 成长弧线 | 从"这份工作只是过渡"到"这里才是我该在的地方"。从用专业保护自己，到敢于暴露真实的脆弱 |

## 固定配角（六人组）

### 方小棠（闺蜜 / 室友 / 徒弟）

| 维度 | 设定 |
|------|------|
| 年龄 | 26 岁 |
| 外貌 | 158cm，圆脸微胖，两个酒窝。骑手服永远大一号，头盔戴不正 |
| 前史 | 中专毕业，农村来城里。做过厂妹、服务员、奶茶店店员。送外卖因为自由——"不用看老板脸色，骑车吹风也挺好" |
| 性格 | 憨厚、容易紧张、运气奇差（下雨必摔、奇葩单必接、红绿灯永远和她作对）。但认死理——对朋友比对谁都好，一旦支棱起来谁都不怂 |
| 与苏冉关系 | 苏冉带她入行。她把苏冉当亲姐。两人合租。苏冉试图教她谈判技巧，她是全世界最差的学生——但偶尔歪打正着 |
| 喜剧功能 | **好心办坏事**——越帮越忙的倒霉体质。苏冉教的每句话她都用错，但错的姿势清奇，结果经常出人意料地有效 |
| 口头禅 | "冉姐救命——""冉姐我是不是又搞砸了——""但是他说——" |
| 成长弧线 | 从"什么都靠冉姐"到关键时刻自己站出来。最软的那个人，在最关键的时候最硬 |

### 秦朗（前拳击手 / 竞争对手 → 暧昧对象）

| 维度 | 设定 |
|------|------|
| 年龄 | 30 岁 |
| 外貌 | 182cm，运动员体型，骑手服穿得像运动品牌广告。话少，面无表情，但眼神很准 |
| 前史 | 省拳击队退役，因手腕伤。家里开面馆的，父亲去世后店关了，欠了一笔债。送外卖因为来钱快、门槛低 |
| 性格 | 话少到令人发指。但每句话都精准——要么是精准的观察，要么是精准的吐槽。行动派，不会安慰人，但会在你需要的时候默默把事办了 |
| 与苏冉关系 | 一开始是竞争对手（争月度单王）。苏冉用脑，他用直觉，互相看不惯。渐渐发现配合默契——苏冉谈 90% 用嘴，剩下 10% 秦朗往那一站就行。感情线是慢炖，不狗血 |
| 喜剧功能 | **面瘫吐槽机**。极度省话 + 开口必切中要害。和苏冉的"过度专业"形成反差——苏冉分析三分钟，秦朗三个字总结完 |
| 口头禅 | "嗯。""走吧。""……"（沉默也是一种台词） |
| 成长弧线 | 从"一个人扛"到学会让人走进来。从只用拳手解决问题，到学会用另一种方式保护人 |

### 老王（站长 / 前退伍军人）

| 维度 | 设定 |
|------|------|
| 年龄 | 45 岁 |
| 外貌 | 中等身材，永远穿一件洗到发白的迷彩 T 恤。头发剩一半，表情永远像在查内务 |
| 前史 | 在部队待了 12 年，副营级转业。战友介绍接管了这个骑手站，把兵那套带过来——早会点名、迟到罚俯卧撑 |
| 性格 | 刀子嘴豆腐心。"少跟我来这套"是他的万能回复。但实际上每次苏冉提建议，他骂完之后会偷偷执行。护犊子——自己可以骂，外人不能碰 |
| 喜剧功能 | **体制内的叛逆者**。代表"规则"，但每次都被苏冉说服去打破规则。嘴上骂苏冉"你以为你还在公安局啊"，遇到麻烦第一个来找她商量 |
| 口头禅 | "少跟我来这套。""你行你来。""……那你说怎么办。"（三句话往往在 10 秒内说完） |
| 成长弧线 | 从"按规矩办事最安全"到"规矩是死的，人是活的" |

### 何仔（大学生兼职骑手 / 流量担当）

| 维度 | 设定 |
|------|------|
| 年龄 | 20 岁 |
| 外貌 | 瘦高，锅盖头，永远戴 AirPods。骑手服上别着一个手机支架，边送外卖边直播 |
| 前史 | 大二，新媒体专业。送外卖不是为了钱（虽然也没钱），是为了做内容——"骑手日常"系列在短视频平台 3 万粉 |
| 性格 | 社交牛逼症。嘴比脑子快。觉得老一辈骑手太严肃，"送个外卖而已又不是打仗"。但他真诚——说错话会道歉，看到不公平会第一个冲上去 |
| 喜剧功能 | **Z 世代解构者**。用互联网角度看一切。苏冉的谈判在他嘴里变成"姐你这是 PUA 啊，但 PUA 得好"。直播视角提供另一种叙事角度 |
| 口头禅 | "家人们谁懂啊——""姐你这个操作我录了啊。""弹幕说你说得对。" |
| 成长弧线 | 从"送外卖就是个内容选题"到真正理解这份工作的重量 |

### 胖虎（老骑手 / 江湖百事通）

| 维度 | 设定 |
|------|------|
| 年龄 | 35 岁 |
| 外貌 | 胖，永远乐呵呵的。骑手服扣子只能扣两颗。认识全城三分之一的保安和一半的小区物业 |
| 前史 | 在这个片区送了 8 年外卖。之前做过保安、跑过滴滴、摆过地摊。江湖经验丰富到令人发指——"这一行所有奇葩事我都见过，没见过的一般不会发生" |
| 性格 | 随和、豁达、嘴碎但句句是真理。不主动说教，但谁问他都给答案。是站里的"生活百科全书"——从"怎么应对顾客差评"到"哪个小区有近道"到"和老婆吵架怎么哄" |
| 喜剧功能 | **过来人旁白**。每集在骑手站的"尾声"环节出现，一句话总结苏冉今天经历的荒唐事。他的经验之谈本身就是段子——"你这算什么，我当年给殡仪馆送过餐，顾客给了差评说面坨了。" |
| 口头禅 | "你这算什么，我当年——""人生嘛。""懂的自然懂。" |
| 成长弧线 | 看似没变化，但有他在，大家心里就踏实。他是骑手站的定海神针 |

### 苏冉妈妈（远程角色 / 血脉压制）

| 维度 | 设定 |
|------|------|
| 年龄 | 55 岁 |
| 设定 | 退休小学教师。典型中国母亲——爱唠叨、爱攀比、爱操心。完全不知道女儿在送外卖，以为苏冉还在"做咨询"（苏冉的谎言） |
| 出场方式 | **前期只通过电话出现**。每次通话都是一场戏——苏冉能用谈判技巧搞定全世界，唯独搞不定自己妈。因为妈妈不用任何技巧，只用"你吃了吗""你什么时候找对象""你那工作到底干什么的"三连击 |
| 喜剧功能 | **专业的天敌**。谈判专家最无力的时刻——对面的人不生气、不激动、就是源源不断地输出关心。你没法"降级"一个温柔的人 |
| 暗线 | 妈妈迟早会发现真相。那将是这一季最重的一场戏 |
| 口头禅 | "你吃了没？""你看人家张阿姨的女儿——""你那个咨询到底是咨询什么的？" |
