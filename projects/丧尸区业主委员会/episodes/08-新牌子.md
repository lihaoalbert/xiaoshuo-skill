# 第 8 集：新牌子

[场景：锦绣花园大门口 — 早晨]

[周铁梅在门口挂了一块新牌子。不是物业公司统一配发的那种蓝色铁牌。是木板。手写的。字是王老师写的——他练过硬笔书法，但歪歪扭扭的，因为木板上不好写。上面写着——]

「锦绣花园人类文明保护区
对外开放时间：有人敲门的时候
物业经理：周铁梅
全体业主共同管理」

[陈主任站在旁边。看了很久。然后——]

陈主任：这个——"共同管理"——谁同意了？

周铁梅：你不是在墙上签名了吗。

陈主任：(嘴张了又合。然后——) ...少给我来这套。

[周铁梅看向他。那个表情——和世界上所有物业经理面对投诉时的表情一样。但有了一点点不一样的东西。可能是笑。]

[林姐带着两只母鸡来看牌子。母鸡对新木头表现出职业性的兴趣。小夫妻在研究"这个牌子的 UI 可以优化"。王老师在算木板的承重——"如果刮风的话——" 老周站在牌子旁边。用手扶了一下。稳了。]

[那对带婴儿的夫妻也来了。婴儿已经不哭了——林姐的母乳管够了。女人看着牌子，哭了。男人没哭——但他把婴儿抱得更紧了。]

[墙外。远处。一个丧尸慢慢走过。它看了一眼牌子。什么都没看懂。走了。]

[周铁梅回到值班室。老周倒了杯茶。桌上摊着登记本。]

周铁梅：(打开登记本。翻到最后一页。写了一个新名字——「未知」。状态——「待登记」。) 这块牌子挂上去——会有更多人敲门的。

老周：嗯。

周铁梅：可能会有坏人也敲门。

老周：嗯。

周铁梅：(把笔放下) 那你站在门口。

老周：...嗯。

[他喝了口茶。没说话。但他的姿势——和末世第一天关上大门时的姿势一样。稳。不准备去任何地方。]
