# 第 3 集：种菜 App

[场景：锦绣花园楼顶 — 下午]

[小赵和小钱——1 号楼的 Z 世代小夫妻——在楼顶开垦了一片菜地。泡沫箱、营养土、从小区花坛挖来的蚯蚓。他们还做了一个东西：一张 A3 纸，手绘表格，标题——]

「末世资源分配 App 1.0 版」

[没有网。没有电。App 是用纸做的。]

小赵：(向周铁梅展示) 你看——每个业主一个账号——其实就是名字写在格子里。用水量、鸡蛋分配、柴油存量——全部可视化！

周铁梅：(看了那张纸。看了很久。) 你们以前做什么的。

小钱：我是 UI 设计师。他是产品经理。

周铁梅：...(把纸放下) 那你们应该知道——用户测试。先给陈主任看看。

[陈主任看到"App"的时候差点把纸撕了。"我一个退休干部，你让我用 App？!" 然后小赵说："陈主任，不是手机 App——是纸的。您可以用笔写。" 陈主任拿起笔，在自己的名字旁边填了用水量。字很工整。他满意了。]

[一周后。那张纸被贴在物业门口。每个业主自己去填。林姐用歪歪扭扭的字写了"鸡蛋 ×3"。王老师用公式计算了全小区的卡路里消耗。陈主任——居然是最积极的那个——每天核对数字，谁填错了找他。]

[小钱站在楼顶看她的菜。小赵站在她旁边。]

小钱：我们做了个 App。没网。没手机。只有纸。但它有用。

小赵：比我们在大厂做过的任何产品都管用。

[楼下。周铁梅看着贴满数据的墙。老周站在旁边。]

老周：...挺像回事的。

周铁梅：嗯。(顿了顿) 就是字太丑了。明天让他们用尺子画线。

---

（第 3 集完）
