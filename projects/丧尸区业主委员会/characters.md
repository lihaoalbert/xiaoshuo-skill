## 主要角色

### 周铁梅（女主 / 物业经理）

| 维度 | 设定 |
|------|------|
| 年龄 | 48 岁 |
| 外貌 | 中等身材，齐耳短发（好打理）。永远穿物业制服——她说"制服让人信任"。手腕上一个老式发圈，随时能把头发扎起来干活 |
| 前史 | 在上海高端住宅做了 15 年物业经理。管过 3000 户的小区。处理过的纠纷包括但不限于：楼上钢琴声太大、楼下养了 12 只猫、某业主把公共走廊改成私人鞋柜、一对离婚夫妻为了一只狗的"探视权"在小区门口对峙——最后她出面调解，狗判给了她 |
| 末世降临那天的选择 | 她在值班。监控屏幕上看到街上的人开始跑。她的第一反应不是跑——是按下了小区大门的遥控器。三道闸门。全部落下。然后她用广播系统说："各位业主，请回到自己的单元。物业将对公共区域进行消杀。"——说的好像只是灭鼠 |
| 性格 | **外在**：冷静、高效、不废话。能同时处理三件事：安抚恐慌的业主、统计库存物资、安排巡逻。**内在**：把小区当成自己的孩子。最大的恐惧不是丧尸——是看到自己保护的人互相伤害。十五年物业经理处理过无数投诉，总结出一条：人性比丧尸可怕多了 |
| 核心喜剧引擎 | 用物业管理的逻辑治理末世。丧尸围城——先开业主大会。有人想硬闯小区——填来访登记表。吵架——发整改通知书。她的世界观是："不管外面什么情况，这里面按规矩来。" |
| 口头禅 | "请填写申请表。""这事不归物业管——但现在物业什么都得管。""你的物业费呢？"——物业费现在是子弹和罐头 |
| 成长弧线 | 从"秩序就是一切"到"有时候规则需要打破，但人心的秩序比规则更重要" |

### 老周（保安队长 / 沉默的右手）

| 维度 | 设定 |
|------|------|
| 年龄 | 50 岁 |
| 外貌 | 退伍军人。话比秦朗还少。走路没声音。站在那就像一堵墙 |
| 前史 | 在部队待到退役。老婆跟人跑了。没什么朋友。周铁梅是他见过的最靠谱的人 |
| 性格 | 不问为什么。只问"怎么做"。小区里流传一个说法——老周年轻时一个人赤手空拳制服过三个醉酒闹事的业主。老周不否认 |
| 与周铁梅关系 | 她是大脑，他是手。她做决定，他执行。从不质疑。唯一一次争吵——她想拿仅剩的药品去换邻居大妈需要的胰岛素。他说"保不住"。她说"你听我的"。他沉默了一分钟。然后去拿车钥匙。 |
| 喜剧功能 | 全场唯一的正常人。其他人都在惊慌/吵架/搞笑时，老周站在旁边。偶尔冒出一句精准到可怕的观察 |
| 口头禅 | "嗯。""我去。""......"(沉默是一种回答) |

## 核心冲突模型

每集 = 一个"业主投诉"的末世版本：

| 投诉类型 | 末世版本 | 笑点来源 |
|---------|---------|---------|
| 噪音投诉 | 3 号楼的发电机太吵 | 那是唯一一台发电机 |
| 违建投诉 | 5 号楼阳台的鸡笼 | 那是唯一的蛋白质来源 |
| 停车纠纷 | 唯一的越野车该归谁用 | "业主大会投票"——结果所有人都投了自己 |
| 公共区域占用 | 有人在大堂晒腊肉 | 那是 5 栋林姐的。用的是丧尸进不来的唯一一块阳光 |

---

## 丧尸设定

- 丧尸不是主角。丧尸是背景——就像《僵尸肖恩》里的僵尸是生活方式的隐喻
- 偶尔有几集丧尸"参与"剧情——但永远是通过人类角色反应来制造喜剧，不是靠丧尸本身
- 丧尸不会被"消灭"。它们会一直在围墙外——就像生活中的那些无法解决的麻烦
