# 后妈 — 角色档案

> 状态：待讨论确认。

## 主要角色

### 顾安然（女主 / 离婚律师 → 继母）

| 维度 | 设定 |
|------|------|
| 年龄 | 32 岁 |
| 外貌 | 168cm，永远穿得很好看但不是刻意打扮——职业习惯。法庭上穿西装，在家穿真丝睡衣。气质偏锋利，笑起来才柔软 |
| 学历 | 中国政法大学 法学硕士 |
| 前职业 | 专打高冲突离婚案的律师。在行业内的标签是"男方律师"——她帮男人打赢了 47 场离婚官司，胜诉率 87%。不是因为她认同男人——是因为她认为"法律面前人人平等，包括你不喜欢的人" |
| 为什么转行 | 最后一场官司的当事人——一个全职妈妈。因为顾安然的辩护，她几乎净身出户。顾安然赢了。然后去厕所吐了十分钟。"法律上的赢，不等于对的赢。" |
| 与宋则明的相遇 | 他是那场官司对方当事人的朋友。官司结束后，他来找她——不是来骂她，是来还她落在法庭的一支笔。"我觉得你不是坏人。但你在帮坏人做事。" 她从来没听过这样说话的人 |
| 性格 | **外在**：冷静、逻辑缜密、嘴毒。能在三句话内找到对方的逻辑漏洞。笑点很高，不轻易被逗笑。**内在**：极度渴望被爱但不会表达。用专业武装自己，因为卸下专业之后她不知道自己是好人还是坏人 |
| 核心矛盾 | 一个帮男人打赢了 47 场离婚官司的女人，现在要做 15 岁女孩的后妈。而这个女孩比她更聪明、更有逻辑、更会"审判"人 |
| 口头禅 | "这不是一个感情问题。这是一个逻辑问题。" |
| 喜剧锚点 | 用法律思维处理家庭生活——给继女写"共同生活协议"、用证据链反驳婆婆的指控、把餐桌对话变成交叉质询 |
| 成长弧线 | 从"法律即正义"到"正义不只在法律里" |

### 宋知夏（继女 / "公诉人"）

| 维度 | 设定 |
|------|------|
| 年龄 | 15 岁，初三 |
| 外貌 | 165cm 还在长。妈妈的眼睛，爸爸的嘴。穿衣风格在"装大人"和"还是小孩"之间反复横跳 |
| 前史 | 妈妈三年前病逝。她在那之后变成了"大人"——帮爸爸处理账单、记住亲戚生日、在学校拿第一名。她不信任任何人能照顾好她爸爸——直到顾安然出现 |
| 性格 | 聪明到让成年人害怕。冷静、记性好、擅长观察矛盾。她不是在闹脾气——她做过的功课比顾安然想象的还多。她查过顾安然每一场官司的判决书 |
| 与顾安然关系 | 婚礼上的那句话是第一枪。此后每一天都在"开庭"——用逻辑挑战顾安然的逻辑，用证据质疑顾安然的动机。但这场战争的目的不是赶走她。目的是——确认她不会走 |
| 口头禅 | "我有几个问题想问一下。" |
| 喜剧锚点 | 未成年人的法律素养超过大多数成年人。写"家庭共同生活提案"比顾安然写的还规范。第 3 集给顾安然发了一封律师函——要求她参加家长会 |
| 成长弧线 | 从"我不能让你替代妈妈"到"你可以不替代她，但你可以是你" |

### 宋则明（父亲 / 丈夫 / "证人"）

| 维度 | 设定 |
|------|------|
| 年龄 | 42 岁 |
| 外貌 | 高，瘦，有点驼背——常年伏案画图的后遗症。笑得时候眼角皱纹很深 |
| 职业 | 建筑师。安静地改造老房子，把快要塌的东西修成新的 |
| 性格 | 温和、善良、略微迟钝。不是不关心——是他不知道两个用逻辑对轰的女人之间的战争该怎么介入。他的方式是在她们吵完以后默默放一杯茶 |
| 喜剧功能 | **全场最无辜的被告**。每次都试图当和事佬，每次都被两边同时否定。口头禅："你们说得都对。" 顾安然和宋知夏同时："不行！" |
| 成长弧线 | 从"让她们自己解决"到学会主动介入、用自己的方式连接两人 |

## 配角

| 角色 | 定位 | 功能 |
|------|------|------|
| 外婆（宋知夏妈妈的母亲） | 60 岁，退休教师。住得不远 | 每集的家庭聚会是"陪审团"。她看顾安然的眼神永远像在看一个犯罪嫌疑人的辩护律师 |
| 周律师（顾安然律所合伙人） | 38 岁，男，单身。顾安然唯一的朋友 | "你帮别人打了 47 场离婚官司，你自己的婚姻你怎么辩护？"——每集的吐槽担当 |
| 林溪（宋知夏闺蜜） | 15 岁，同班同学 | 宋知夏的"智囊团"。两人经常密谋"试探顾阿姨"的方案。每次都以失败告终但从不气馁 |
| 宋知夏妈妈 | 仅在照片和闪回中出现 | 不是完美的妈妈。但她温暖。宋知夏记得的全部是她好的一面。顾安然最大恐惧——永远比不过一个"不存在的人" |

## 核心冲突模型

每集 = 宋知夏发起的"诉讼"：
1. 她提出一项指控（"你不关心我爸""你做的饭都是外卖""你和他前妻不一样"）
2. 她提供证据（观察、截图、录音、朋友圈分析）
3. 顾安然必须"辩护"——但不能用律师的方式赢
4. 宋则明是证人、法官、或者搅局者
5. 每集结局不是"谁赢"——是双方都退了一步
