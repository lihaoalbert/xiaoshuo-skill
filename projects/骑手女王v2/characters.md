## 苏冉（女主角 / "骑手女王"）

| 维度 | 设定 |
|------|------|
| 年龄 | 30 岁 |
| 外貌 | 165cm，中上长相，皮肤不算白（常年风吹日晒但很干净）。扎马尾戴头盔是"苏师傅"，下班放下头发换掉工服判若两人。工服没有女款——最小号也大一号，腰上别了两个夹子。 |
| 学历 | 中国人民公安大学 犯罪心理学硕士 |
| 前职业 | 市公安局危机谈判组，最年轻的谈判组长。处理过跳楼、劫持、家暴对峙。绰号"冰箱"——不管对面多炸，她永远是全场最冷静的人。 |
| 离职原因 | 最后一次任务：她救下了所有人。第二天她自己签了离职申请。"再待下去，我会变成我不认识的人。"同事不理解。领导不理解。她妈更不理解。 |
| 入行契机 | 面试37家公司全挂——"学历太高了不敢用"、"谈判专家？我们又不是黑社会"。饿了么写着"月入过万，学历不限"。房贷还剩23年。 |
| 性格 | **外在**：嘴贫、脸皮厚、什么场面都稳得住。**内在**：要强、不服输、习惯一个人扛。能化解所有人的冲突，唯独不处理自己的创伤。 |
| 核心能力 | 读人（30秒判断人格类型）、控场（语速/节奏/沉默的精准使用）、降级（让对方在不知不觉中自己把情绪降下来） |
| 口头禅 | "我们先明确一下各自的诉求。" （对谁都像在谈判——包括对五岁小孩和小区保安） |
| 喜剧锚点 | **过度专业 × 鸡毛蒜皮**——用解救人质的专注度处理"外卖少放香菜"级别的纠纷。但她的专业经常碰壁。 |
| 能力碰壁清单（每集至少一个碰壁） | ① 五岁小孩不听逻辑 ② AI客服不吃谈判策略 ③ 老妈电话让所有技巧清零 ④ 物业保安不是敌人——她的"威胁评估"框架不适用于单纯态度差的人 ⑤ 菜场砍价——她对价格没概念，因为以前不需要砍价 ⑥ 方小棠——全世界最差的学生，教什么毁什么 |
| 致命弱点 | 1) 妈妈——所有谈判技巧在血脉压制面前清零 2) 不会求助——能帮所有人搞定问题，唯独不会让人帮自己 3) 对"真诚"过敏——遇到完全不需要谈判的正常人反而手足无措 |
| 成长弧线 | 从"这份工作只是过渡"到"这里才是我该在的地方"。从用专业保护自己，到敢于暴露真实的脆弱。从"我是谈判专家苏冉"到"我是骑手苏冉，工号0427"。 |
| 对妈谎言 | 跟妈说在做"管理咨询"。"帮企业优化流程那种。"妈在电话里问"那什么时候调回来"——她回"快了。快了。"窗外是骑手站的塑料棚顶。这是她唯一搞不定的谈判。 |

## 方小棠（闺蜜 / 室友 / 徒弟）

| 维度 | 设定 |
|------|------|
| 年龄 | 26 岁 |
| 外貌 | 158cm，圆脸微胖，两个酒窝。骑手服永远大一号（和苏冉一样的问题——工服没女款），头盔戴不正。 |
| 出身 | 重庆秀山农村。中专毕业——学的会计，但Excel只会填数字。来城里做过厂妹、服务员、奶茶店店员。送外卖因为自由——"不用看老板脸色，骑车吹风也挺好。" |
| 性格 | 憨厚、容易紧张、运气奇差（下雨必摔、奇葩单必接、红绿灯永远和她作对）。但认死理——对朋友比对谁都好。一旦支棱起来谁都不怂。 |
| 与苏冉关系 | 苏冉带她入行。她把苏冉当亲姐。两人合租——方小棠负责泡面，苏冉负责点外卖。苏冉试图教她谈判技巧，她是全世界最差的学生——但偶尔歪打正着。 |
| 喜剧功能 | **好心办坏事**——越帮越忙的倒霉体质。苏冉教的每句话她都用错，但错的姿势清奇，结果经常出人意料地有效。 |
| 个人细节 | 每个月苏冉痛经，她会提前往苏冉包里塞暖宝宝——"冉姐你别硬撑。"她自己的创可贴永远不够用（因为每天都有新的擦伤），但苏冉受伤时她能从包里翻出三种不同尺寸的创可贴。 |
| 口头禅 | "冉姐救命——""冉姐我是不是又搞砸了——""但是他说——" |
| 成长弧线 | 从"什么都靠冉姐"到关键时刻自己站出来。最软的那个人，在最关键的时候最硬。 |

## 秦朗（前拳击手 / 竞争对手 → 暧昧对象）

| 维度 | 设定 |
|------|------|
| 年龄 | 30 岁 |
| 外貌 | 182cm，运动员体型。骑手服穿得像运动品牌广告。话少，面无表情，但眼神很准——他看出苏冉的站姿不对的第一天，就知道她不是普通骑手。他没问。 |
| 前史 | 省拳击队退役，因手腕伤。家里开面馆的，父亲去世后店关了，欠了一笔债。送外卖因为来钱快、门槛低。 |
| 性格 | 话少到令人发指。但每句话都精准——要么是精准的观察，要么是精准的吐槽。行动派，不会安慰人，但会在你需要的时候默默把事办了。 |
| 与苏冉关系 | 一开始是竞争对手（争月度单王）。苏冉用脑，他用直觉，互相看不惯。渐渐发现配合默契——苏冉谈90%用嘴，剩下10%秦朗往那一站就行。感情线是慢炖，不狗血。 |
| 喜剧功能 | **面瘫吐槽机**。极度省话 + 开口必切中要害。和苏冉的"过度专业"形成反差——苏冉分析三分钟，秦朗三个字总结完。 |
| 个人细节 | 右手腕有一道疤——不是打架的，是退役前最后一场比赛，韧带撕裂。平时戴护腕盖着。下雨天手腕会疼。方小棠注意到他下雨天不太用右手——"秦朗哥你手怎么了？"他回："没事。"然后换了左手提外卖。走到门口停了一下。转身。帮方小棠把她的外卖也提上了。还是一句话没说。每天收工后他会绕路经过一家关张的面馆——不是想吃面，是闻一下味道。他爸的面馆就是这个味道。他从来没跟任何人提过。 |
| 口头禅 | "嗯。""走吧。""……"（沉默也是一种台词） |
| 成长弧线 | 从"一个人扛"到学会让人走进来。从只用拳手解决问题，到学会用另一种方式保护人。 |

## 老王（站长 / 前退伍军人）

| 维度 | 设定 |
|------|------|
| 年龄 | 45 岁 |
| 外貌 | 中等身材，永远穿一件洗到发白的迷彩T恤。头发剩一半，表情永远像在查内务。 |
| 前史 | 在部队待了12年，副营级转业。战友介绍接管了这个骑手站，把兵那套带过来——早会点名、迟到罚俯卧撑、罚款单用手写——"App系统自动开的罚款不算。我开的才算。" |
| 性格 | 刀子嘴豆腐心。"少给我来这套"是他的万能回复。但实际上——每次苏冉提建议，他骂完之后会偷偷执行。护犊子——自己可以骂，外人不能碰。 |
| 喜剧功能 | **体制内的叛逆者**。代表"规则"，但每次都被苏冉说服去打破规则。嘴上骂苏冉"你以为你还在公安局啊"，遇到麻烦第一个来找她商量。 |
| 个人细节 | 他桌上有两个杯子。一个旧的搪瓷杯——部队发的，茶垢积了三层，从来不洗——"洗了没味。"另一个新的保温杯——方小棠送的，因为"王站你老喝凉的胃会坏。"两个杯子都摆在桌上。搪瓷杯在上。保温杯在旁边。保温杯从来没被用过。但从来没被收起来。 |
| 口头禅 | "少给我来这套。""你行你来。""……那你说怎么办。"（三句话往往在10秒内说完） |
| 成长弧线 | 从"按规矩办事最安全"到"规矩是死的，人是活的"。从偷偷帮苏冉到公开站她——"这是苏冉的方案。我改的。署的她名字。递上去了。" |

## 何仔（大学生兼职骑手 / 流量担当）

| 维度 | 设定 |
|------|------|
| 年龄 | 20 岁（已成年，大学生） |
| 外貌 | 瘦高，锅盖头，永远戴AirPods。骑手服上别着一个手机支架，边送外卖边直播。 |
| 前史 | 大二，新媒体专业。送外卖不是为了钱（虽然也没钱），是为了做内容——"骑手日常"系列在短视频平台3万粉。他爸妈不知道他在送外卖。"我说我在做田野调查。社会学的。他们觉得可高级了。" |
| 性格 | 社交牛逼症。嘴比脑子快。但真诚——说错话会道歉，看到不公平会第一个冲上去。 |
| 喜剧功能 | **Z世代解构者**。用互联网角度看一切。苏冉的谈判在他嘴里变成"姐你这是PUA啊，但PUA得好。"直播视角提供另一种叙事角度。 |
| 直播梗刷新规则 | 每个使用直播的场景必须有新角度——弹幕内容的具体化（"有条弹幕问：情绪价值能选秦朗配送吗"）、观众互动反转、意外事件——而非重复"家人们/弹幕炸了"。 |
| 个人细节 | 他第一次被苏冉帮了之后，直播间标题从"骑手日记day47"改成了"跟女王学做人day1"。弹幕有人说标题太舔。他回："你们不懂。她能用谈判技巧让一个要给我差评的人改成了五星好评+打赏。这叫实力。我学。我录。我发。"他的AirPods永远有一只没电——因为直播耗电快，充电盒也经常忘带。

他直播时镜头总往方小棠那边偏——角度刚好够拍到她的侧脸和永远戴不正的头盔。他收藏了她每次说"冉姐救命——"的音频。弹幕有人问"主播你是不是喜欢那个头盔歪的小姐姐"，他回："我在做田野调查。"弹幕："社会学的？"他："对。社会学。" |
| 口头禅 | "姐你这个操作我录了啊。""弹幕说你说得对。" |
| 成长弧线 | 从"送外卖就是个内容选题"到真正理解这份工作的重量。从"我在做田野调查"到"我在送外卖——跟我爸妈也是这么说的。" |

## 胖虎（老骑手 / 江湖百事通）

| 维度 | 设定 |
|------|------|
| 年龄 | 35 岁 |
| 外貌 | 胖，永远乐呵呵的。骑手服扣子只能扣两颗。永远在剥花生——花生是自带的，装在工服口袋里。 |
| 前史 | 这个片区送了8年外卖。之前做过保安、跑过滴滴、摆过地摊、开过烧烤摊——后来烧烤摊被收了（其实是自己没办食品证，城管照规矩来。但那时候他不知道办证要什么材料，只知道交钱。"算了，跑外卖吧。"）认识全城三分之一的保安和一半的小区物业。 |
| 性格 | 随和、豁达、嘴碎但句句是真理。不主动说教，但谁问他都给答案。是站里的"生活百科全书"——从"怎么应对顾客差评"到"哪个小区有近道"到"和老婆吵架怎么哄"——"哄什么哄。多做一顿饭。她爱吃我做的蒜蓉西兰花。" |
| 喜剧功能 | **过来人旁白**。每集在骑手站的"尾声"环节出现，一句话总结苏冉今天经历的荒唐事。他的经验之谈本身就是段子——"你这算什么，我当年给殡仪馆送过餐，顾客给了差评说面坨了。我说大哥，人家在殡仪馆吃面——面坨不坨已经不是重点了。重点是面。" |
| 个人细节 | 花生的数量就是他的情绪指数。剥得越快→越放松。花生停在半空→出事了。花生壳攒在口袋里→他在想事情。有一次老王罚款太狠，胖虎从早会到傍晚一颗花生没剥——那天站里所有人都不敢跟他说话。后来苏冉问了才知道：他不是生气。他是在想怎么帮一个被罚的新骑手——"那孩子比我当年还愣。我当年好歹会认路。" |
| 口头禅 | "你这算什么，我当年——""人生嘛。""懂的自然懂。" |
| 成长弧线 | 看似没变化，但有他在，大家心里就踏实。他是骑手站的定海神针。 |

## 苏冉妈妈（远程角色 / 血脉压制）

| 维度 | 设定 |
|------|------|
| 年龄 | 55 岁 |
| 设定 | 退休小学教师。典型中国母亲——爱唠叨、爱攀比、爱操心。完全不知道女儿在送外卖，以为苏冉还在"做咨询"。 |
| 出场方式 | **前期只通过电话出现**。每次通话都是一场戏——苏冉能用谈判技巧搞定全世界，唯独搞不定自己妈。因为妈妈不用任何技巧，只用"你吃了吗""你什么时候找对象""你那工作到底干什么的"三连击。 |
| 喜剧功能 | **专业的天敌**。谈判专家最无力的时刻——对面的人不生气、不激动、就是源源不断地输出关心。你没法"降级"一个温柔的人。 |
| 暗线 | 妈妈迟早会发现真相。那将是这一季最重的一场戏。但处理方式不是苦情——是苏冉终于说出口的那一刻，她发现自己不是在认输，是在说真话。 |
| 口头禅 | "你吃了没？""你看人家张阿姨的女儿——""你那个咨询到底是咨询什么的？" |

## 身份暴露设计原则（平台合规保障）

苏冉的前职业身份可能被意外发现。每次的处理方式必须遵循以下规则：

| ❌ 禁止（爽文模式） | ✅ 执行（喜剧/反高潮模式） |
|-----|-----|
| 对方震惊："你以前是谈判专家？！" | 对方："哦。那你以前工资多少？" |
| 苏冉展示技能，众人崇拜 | 苏冉展示技能，何仔直播："家人们这个技巧能不能用来砍价？"——弹幕全在讨论砍价，没人关心她的过去 |
| 身份揭晓 = 高潮 | 身份揭晓 = 尴尬（"那你为什么在这送外卖？"——苏冉不知道怎么回答） |
| 身份是秘密武器 | 身份是尴尬记忆——她不想提不是因为"秘密"，是因为自己还没完全接受"我从那里离开了" |
