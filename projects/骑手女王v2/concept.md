# 骑手女王 v2 — 概念提案

## 一句话

一个前谈判专家的新战场：用解救人质的专注度，处理少放了香菜的麻辣烫。

## 概念场景（三句话让读者看到这部剧）

苏冉蹲在小区门口。一个五岁男孩嚎啕大哭——"我不要吃青菜！！"

她的蹲姿距离、身体45度角、双手可见——教科书式的危机谈判站位。围观大妈以为她只是个哄孩子的骑手。

"你的诉求我听到了。青菜的敌人不是你。"她从外卖袋里掏出炸鸡。"但我们谈判的第一步——是先咬一口这个。"

男孩不哭了。开始谈判。

苏冉：(内心 OS) 对方代表年龄约五岁。谈判经验：零。威胁等级：——比我上一次的人质谈判高。那次的人质至少讲道理。

## 核心设定

苏冉，30岁。前市公安局危机谈判组组长。在一次成功谈判——她救下了所有人——之后，她自己签了离职申请。

"再待下去，我会变成我不认识的人。"

同事不理解。领导不理解。她妈更不理解。"你疯了吗？公务员！铁饭碗！你张阿姨的女儿考了三年都没考上！"

房贷还剩23年。面试了37家公司。"学历太高了我们不敢用"、"谈判专家？我们又不是黑社会"。

饿了么招聘广告写"月入过万，学历不限"。她点进去。填了表。第二天领了黄色工服。

这份工作本来是过渡。但一个月后，她发现——这里的人需要她。不是需要她的专业执照。是需要她。方小棠被差评威胁时拉着她的衣角。何仔被多人行挖角时看着她等她说话。老王嘴上骂"你少给我来这套"然后默默把她的名字写在总部文件上。

那张谈判桌，她不回去了。因为这里有了另一张桌子。塑料的。放在骑手站门口。上面有六个人的茶杯印。

## 核心喜剧引擎

**过度专业 × 鸡毛蒜皮**
用解救人质的专注度处理"外卖少放香菜"级别的纠纷。
**但记住——她的专业经常碰壁。**
五岁小孩不听BATNA。AI客服不吃谈判策略。老妈的电话能让所有技巧清零。

## 她的身体记得（不可替代的女性经验）

- 每个月那几天，痛到脸色发白还要爬六楼。方小棠往她包里塞暖宝宝——"冉姐你别硬撑。"
- 跑晚单时把头盔压得最低，工服拉链拉到下巴——不是怕冷。是怕被人认出来是个女的。
- 工服没有女款。最小号也大一号。她在腰上别了两个夹子。
- 有顾客开门看到她，"哟，骑手姐姐你这么拼啊"——语气里的打量比差评更让人不舒服。

## 她说不出口的东西（情感暗线）

她妈不知道她在送外卖。她跟妈说在做"管理咨询"。"帮企业优化流程那种。"

妈在电话里说"你张阿姨女儿又升职了。你那个咨询——到底咨询什么的？"

苏冉：就是帮人解决问题。跟以前差不多。妈：那什么时候调回来？

苏冉把手机拿远了一点。窗外是骑手站的塑料棚顶。

"快了。快了。"

这是她唯一搞不定的谈判。

## v2 关键改动（基于 v1 审查反馈）

1. **苏冉的离开叙事重构**：原版"上级指挥失误→推出来背锅"改为"成功谈判后的主动辞职"。消除平台"抹黑公职人员"红线风险，同时让苏冉的选择更有力量——她不是被抛弃的，她是自己走的。她的失去不是来自外部的惩罚，而是来自内部的"我不再认识自己了"。

2. **内心 OS 去术语化**：保留苏冉的分析思维框架，去掉BATNA/威胁等级等专业黑话。用更口语化、更像"人在想事情"的方式呈现。同时建立"视觉行为词典"——苏冉不需要台词就能外化专业习惯的动作清单——作为去掉术语后的钩力替代方案。

3. **能力碰壁原则**（平台合规核心设计）：苏冉的谈判技能在新环境中**经常失效**。小孩不听逻辑。系统不给人情。妈妈让所有专业清零。她不是来碾压的——她是来学习的。任何她过去身份被意外发现的戏份，设计为"尴尬/被调侃"而非"众人膜拜"。这是区分"喜剧"和"爽文"的根本原则。

4. **第二幕提前规划篇幅**：原版第二幕(9-16集)全部低于1500字标准。v2 在分集大纲阶段就为每集规划足够的喜剧节拍和B线。

5. **何仔直播梗刷新机制**：每个使用直播的场景需要新的角度——弹幕内容、观众互动、意外事件——而非重复"家人们/弹幕炸了"。

6. **开场钩子前置设计**：Ep1的第一帧必须包含一个不可误读为任何其他类型的喜剧视觉信号。用视觉并置而非台词确立荒诞底色。后续每集开场在分集大纲阶段即预设钩子策略。

## 视觉行为词典（苏冉不需要台词的外化动作）

| 行为 | 画面 | 暴露什么 |
|------|------|---------|
| 进入任何空间先扫一圈 | 她的眼睛在动，头没动 | 本能地评估"出口、威胁、人员分布" |
| 对距离的精确感知 | 蹲下时和对方保持的45度角和120cm距离 | 谈判站位肌肉记忆 |
| 听到争吵声的头部微转 | 旁边有人吵架，她的头转了15度——刚好够听到关键信息 | 听觉注意力分配是训练过的 |
| 手的位置 | 面对激动的人时，她的手永远在对方视线内、掌心可见 | 降低对方威胁感的职业习惯 |
| 沉默的使用 | 对方在吼，她不回嘴。她在等——等那个"吼完之后往下掉的瞬间" | 谈判专家最核心的技能不是说话，是知道什么时候不说话 |

## 项目定位

| 维度 | 定位 |
|------|------|
| 类型 | 都市轻喜剧（A级扶持） |
| 调性策略 | 第一帧即喜剧——用视觉并置而非台词确立荒诞底色。杜绝悬疑/苦情/逆袭类型误读 |
| 集数 | 24集 × 3幕（8集/幕） |
| 每集字数 | 1500-2800字（第二幕不低于1800字硬底线） |
| 目标读者 | 20-35岁中国都市女性 |
| 平台合规 | A级都市轻喜，无绝对禁止项触发 |
| 核心主题 | 从"这份工作只是过渡"到"这里才是我该在的地方" |

## 三幕结构

| 幕 | 集数 | 主题 | 核心冲突 |
|----|------|------|---------|
| 第一幕 | 1-8 | "我不是骑手" | 苏冉隐藏过去→被迫使用技能（且碰壁）→第一次被团队需要 |
| 第二幕 | 9-16 | "我们是骑手" | 多人行业务竞赛→团队面临拆散→女王配送品牌建立（内部优化，非对抗平台） |
| 第三幕 | 17-24 | "骑手女王" | 妈妈发现真相→苏冉面对自己→她选择留下（不是因为无处可去，是因为这里需要她） |

## 角色六人组（骨架，待 Phase 2 展开）

- **苏冉**（女主）：前谈判专家，主动辞职，外卖骑手。房贷23年。对妈撒谎。
- **方小棠**（闺蜜/室友）：农村来城，中专毕业。憨厚倒霉体质。苏冉的第一个徒弟，但全世界最差的学生。
- **秦朗**（前拳击手/暧昧对象）：话少到令人发指，每句都精准。苏冉分析三分钟，他三个字总结完。往她身后一站，比她解释一百句都有用。
- **老王**（站长/前退伍军人）：刀子嘴豆腐心。口头禅"少给我来这套"——说完偷偷帮。
- **何仔**（大学生/直播骑手/20岁）：Z世代视角。用互联网解构一切。第一次被苏冉帮了之后，直播间标题改成了"跟女王学做人"。
- **胖虎**（老骑手/江湖导师）：片区8年。口头禅"你这算什么，我当年——"。剥着花生，说着真理。
