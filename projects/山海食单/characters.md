## 主要角色

### 姜晚（女主 / 厨师）

| 维度 | 设定 |
|------|------|
| 年龄 | 25 岁 |
| 外貌 | 160cm，圆脸，手上有好几道刀疤和烫伤——厨师的手。不漂亮但笑起来让人想吃饭 |
| 学历 | 烹饪职校毕业。师傅说她是"最差的学生"——因为她不肯按菜谱来 |
| 前史 | 在成都一家老牌川菜馆做了三年。被开除的原因：她改良了一道有六十年历史的招牌菜。老师傅说她"不尊重传统"。她觉得自己只是想让菜更好吃 |
| 性格 | 固执、嘴硬、对食材有近乎偏执的直觉。闻到食材就知道该放什么。不擅长和人相处，但擅长和锅相处——因为锅不会反驳她 |
| 核心喜剧引擎 | 一个学川菜的厨师，要复刻周朝的菜——而周朝的菜谱上写的是"醢""臡""淳熬"这种连字都不认识的东西 |
| 口头禅 | "这道菜不对。" |
| 成长弧线 | 从"传统是枷锁"到"传统是你可以对话的人，不是你需要打倒的敌人" |

### 顾野（男主 / 食物史研究者）

| 维度 | 设定 |
|------|------|
| 年龄 | 28 岁 |
| 外貌 | 瘦，戴眼镜，永远背一个帆布包，里面装着古籍和泡面 |
| 前史 | 历史学博士读到第三年退学了。论文写的是"先秦饮食中的祭祀功能"。退学原因：导师说"没有学术价值"。现在在一家小博物馆做策展，周末在 B 站开一个 2000 粉的频道"古人怎么吃饭" |
| 性格 | 社恐、轴、一聊到古代饮食就停不下来。姜晚说他"行走的菜谱翻译器"。但他翻译的菜谱——"醓醢以薦"——姜晚一个字都听不懂 |
| 与姜晚的关系 | 姜晚在他博物馆的"古代厨房"展区里用一口仿制的青铜鼎做了一道菜。展品差点烧了。他本该报警。但尝了一口——眼泪出来了 |
| 喜剧功能 | 姜晚的行动派 vs 顾野的学术派。姜晚："这个东西能吃不？" 顾野："在《周礼》中，这种食材只有在祭祀时配合——" 姜晚："所以能吃不？" |
| 成长弧线 | 从"古籍比活人重要"到"菜是用来吃的，不是用来研究的" |

## 妖怪角色（每集轮换）

### 第 1-3 集妖怪

| 妖怪 | 《山海经》原型 | 困境 | 对应古菜 | 人间故事 |
|------|--------------|------|---------|---------|
| 虎蛟 | "其状鱼身而蛇尾，其音如鸳鸯" | 失去了声音——再也唱不出求偶的歌 | "鱼脍"（周朝生鱼片，用八种酱料） | 一个失声的京剧演员 |
| 讙 | "其状如狸，一目而三尾，其音如叱呼" | 被世人遗忘——再也没有人叫它的名字 | "淳熬"（周朝肉酱盖浇饭） | 一个被子女送去养老院的老人 |
| 当康 | "其状如豚，有牙，其鸣自叫，见则天下大穰" | 再也无法带来丰收——经过的田地都枯萎 | "炮豚"（周朝烤乳猪） | 一个破产的养猪户 |

### 妖怪的设计原则
- 每个妖怪的"病"是人类情感的隐喻
- 每道古菜治愈的不是妖怪的超能力，是它的"心病"
- 姜晚不是用魔法做菜——她是用厨师的直觉理解食材和食客之间的连接

## 配角

| 角色 | 定位 | 功能 |
|------|------|------|
| 花椒 | 姜晚的猫。橘猫。胖。对任何人类食物无动于衷 | 喜剧担当。也是判断菜"对不对"的唯一裁判——花椒肯吃，就是对的 |
| 山海 App | 不是人类角色。只在手机上出现 | 每一集发送一个坐标 + 菜名。没有客服、没有帮助中心、没有隐私协议。姜晚试图卸载过——第二天它自己又出现了 |
